How to work with hammer paths???
1) In Hammer settings add this "$SteamUserDir\NameOfYoursGameFolder" without "..\" in all paths
2) In Hammer settings add this "$SteamUserDir\NameOfYoursGameFolder\NameOfYoursMod" or "SourcemodsFolder\NameOfYoursMod" without "..\hl2" in all paths
(Game Configurations and Build Programs)
3) Add yours .fgds and .exe program.
4) That's All! Enjoy and creating maps for this producte!
